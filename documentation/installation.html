Important Notes

Before you attempt to run the program you will need to have the latest version of java installed, as our solution uses Lambda Expressions introduced in Java 8.

As well as the latest version of Gradle.

Please check this Java Verification Link to determine you have the latest version of java installed.

As well as this Gradle Installation tutorial to help in your gradle installation.

Installing the program:

Thankfully, there is close to no effort in running the program

Simply traverse to the main directory of the program and run "gradle run" to properly run the program.

All tests can be viewed and performed when opeening up the program into an IDE For example: Eclipse.
